# üöÄ 16-Week Roadmap

## **Phase 1 ‚Äî Math & Classic ML (Weeks 1‚Äì4)**

### **Week 1 ‚Äì Linear Algebra**
- Matrix operations, projections  
- Eigenvalues, SVD, PCA  
- **Project:** PCA on a real dataset  

### **Week 2 ‚Äì Probability & Statistics**
- Distributions, variance, covariance  
- Likelihood, MLE  
- Entropy, KL divergence  
- **Project:** Compare accuracy vs log-loss  

### **Week 3 ‚Äì Optimization & Supervised Learning**
- Gradient descent, SGD  
- L1/L2 regularization  
- **Project:** Linear regression from scratch  

### **Week 4 ‚Äì Trees & Evaluation**
- Decision Trees, Random Forests, XGBoost  
- Precision/Recall, F1, ROC, AUC  
- **Project:** Handling class imbalance  

---

## **Phase 2 ‚Äî Deep Learning Fundamentals (Weeks 5‚Äì8)**

### **Week 5 ‚Äì Neural Network Basics**
- MLPs, forward + backward pass  
- Loss functions  
- **Project:** MLP classifier  

### **Week 6 ‚Äì DL Regularization**
- Dropout, BatchNorm, Early Stopping  
- Weight Decay  
- **Project:** Compare regularization strategies  

### **Week 7 ‚Äì CNNs for Vision**
- Convolution, filters, pooling  
- Build a CNN (MNIST/CIFAR-10)  
- **Project:** CNN vs MLP comparison  

### **Week 8 ‚Äì Transfer Learning**
- Pretrained models (ResNet, VGG)  
- Data augmentation  
- **Project:** Fine-tuning ResNet  

---

## **Phase 3 ‚Äî NLP & Engineering (Weeks 9‚Äì12)**

### **Week 9 ‚Äì Classical NLP**
- Bag-of-Words, TF‚ÄìIDF  
- Tokenization, text cleaning  
- **Project:** TF‚ÄìIDF classifier  

### **Week 10 ‚Äì Transformers**
- Self-attention (conceptual)  
- Fine-tune BERT/DistilBERT  
- **Project:** Classical NLP vs BERT  

### **Week 11 ‚Äì Software Engineering for ML**
- Clean repo structure  
- Git, logging, modular code  
- Testing (pytest)  
- **Project:** Refactor earlier project  

### **Week 12 ‚Äì Deployment**
- FastAPI model serving  
- Docker basics  
- **Project:** Build a containerized ML API  

---

## **Phase 4 ‚Äî Specialization & Capstone (Weeks 13‚Äì16)**

Choose one specialization:
- Advanced DL / Generative Models  
- Reinforcement Learning  
- Time-Series Forecasting  
- Recommender Systems  
- Bayesian ML  

### **Week 13 ‚Äì Deep Dive Project**
- VAE / GAN / diffusion  
- DQN / PPO  
- Matrix Factorization  
- Transformer for Time-Series  
- Bayesian Regression  

### **Week 14 ‚Äì Capstone Setup**
- Define the problem  
- Gather data, perform EDA  

### **Week 15 ‚Äì Capstone Modeling**
- Baseline ‚Üí advanced models  
- Experiment tracking (MLflow/W&B)  

### **Week 16 ‚Äì Capstone Completion**
- Clean code  
- Full README  
- Optional: Deploy API
